## 简介

Showtime 是一个开源的屏幕共享工具，专为开发者设计。它提供了简单、安全、高效的屏幕共享解决方案，解决了在使用其他会议软件时可能遇到的延迟高、质量差的问题。

## 特性

* 多用户屏幕共享
* 通过 WebRTC 进行安全传输
* 低延迟 / 高分辨率
* 通过 Docker 或单一二进制文件简单安装
* 集成 TURN 服务器，解决 NAT 穿透问题
